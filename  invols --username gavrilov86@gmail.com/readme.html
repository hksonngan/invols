О программе

Программа предназначена для просмотра файлов с траекториями, сгенерированных комплексом SuperCol. Возможно, в дальнейшем есть смысл открывать не файл, а папку с данными эксперимента... После загрузки траектории появяться в графическом окне (первая траектория ярче остальных). Для примера можно загрузить TrjETubM.dat. Также появиться сечение - плоскость, перпендикулярная оси Z, на которую проектируются траектории (проектируется только последний 'виток' траекторий). Геометрия гиротрона и сетка с осями выводятся всегда (геометрия гиротрона загружается из файла Geom_Gun.dat).

Интерфейс

Главное окно программы поделено на окно для вывода графики (слева) и окно для просмотра статистики и изменения параметров (справа). Вид в левом окне регулируется мышью. В случае "3D режима" можно вращаться вокруг сцены и приближать/удалять её. За центр сцены считается сечение. В случае "2D режима" - менять масштаб и двигать сцену. Режим меняется кнопкой "3D mode" на пенели инструментов. В правом окне две вкладки - статистика и параметры. В статистике, как и вообще в программе, все величины в тех единицах измерения, в каких они были в исходных файлах. Единственное - углы из файлов с треками в градусах считаются. А время и длина - безразмерны. Сетка имеет шаг 1 (в окрестности сечения - 0.1). В нижней части окна (в статусе) выводяться координаты, на которые указывает курсор мыши (только в 2Д режиме).

Статистика

Для каждого из параметров приводится статистика: минимум, максимум и среднее. Статистика проводится по всем итерациям в каждой траектории, т.е. число статистических выборок = число итераций * кол-во траекторий. Причём среднее вычисляется с учётом времени. Например, средняя скорость - это не (v1+...+vn)/n (n - число итераций), а (v1*dt1+...+vn*dtn)/t ( t = dt1 +...+ dtn - время). Таким образом, средняя скорость - это среднее арифметическое скоростей всех электронов в постоянно работающем гиротроне. Аналогично и для других величин в статистике по траекториям. Параметр g - это отношение поперечной скорости электрона к продольной (т.е. | Vxy | / | Vz |).

Статистика для сечения собираестя только для точек траекторий на сечении (точнее, для ближайших точек, с координатой Z < Z-координаты сечения). Т.е. сколько траекторий, столько и статистических выборок. Частота - это обратная величина к длительности последнего перед сечением витка траектории.

Параметры

Можно выводить несколько серий траекторий - выводяться те же исходные траектории, но несколько раз - с различным углом поворота вокруг Oz.

При изменении положения сечения изменения будут и в графическом окне, и в статистике для сечения.
